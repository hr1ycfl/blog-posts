# 搭建 hexo 环境
## 1. 使用 hexo 需要安装 **nodejs** 
> 推荐使用 **nvm** 进行安装

## 2. hexo主题以及静态网站的部署则需要安装 **git**
> git部署时,需要选择仓库[**github**,**coding**]都可以,但是 **github** 不支持百度抓取,所以推荐 **coding**

## 3. 安装好以上插件之后，进行 **hexo** 的安装
>npm install -g hexo-cli //hexo 所需要的脚手架
